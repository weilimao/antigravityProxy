1. 多账号导入与导出支持：
   - 界面上新增了“导入账号”和“导出账号”功能，支持批量导入多个账号的 JSON 配置文件，并提供导入防重与关键信息自动校验（过滤缺失邮箱或 Token 的无效数据）。
   - 支持在账号列表卡片中单独导出该账号对应的 JSON 配置文件。
2. 多账号用量与成本细分统计 (Usage Tracker)：
   - 新增账号用量详情可视化折叠面板，展示各账号的调用次数、Token 吞吐量（输入、输出、缓存）、缓存命中率以及各项支出的细分成本和总成本。
   - 支持按账号展开其调用的各模型的用量和三项成本明细（输入成本、输出成本、缓存成本）。
   - 重构了费率计算模块，支持更精细的输入/输出/缓存分项费率换算与存储。
3. 系统稳定性与体验优化：
   - 针对系统命令（如 Windows 的 certutil 和 macOS 的 security）可能发生挂起卡死导致主程序假死的问题，在 certManager 中引入了 4s 强超时保护与 Dashboard 端的自动重试机制。
   - 在 Antigravity 渠道下，仅在未启用 poolMode（账号池轮询）时激活新账号才自动关闭其他账号，确保轮询机制可以正常运作。
   - 优化了模型统计列表的排序逻辑，优先按照总 Token 消耗降序排列。
