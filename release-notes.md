1. 抓包分析与接口文档一键分析生成功能：
   - 界面上新增了“抓包分析 (PACKETS)”选项卡，以宽屏大屏高利用率布局呈现。左侧实时汇总并通过 Method + Host + Path 对捕获到的 200 成功报文进行去重；右侧清晰呈现完整的请求/响应 Header 和 Body。
   - 对接账号池，允许用户指定 Enabled 账号，一键调用 AI 接口分析报文，自动生成详尽的 API 说明文档。
   - 支持通过系统文件保存对话框，将生成的 Markdown 文档原生导出下载到本地。
2. 深度嵌套字段解析与 JSON 智能压缩优化：
   - 引入了树形字段路径扁平化提取（如 `response.paidTier.availableCredits[].creditType` 路径），并在 AI Prompt 中作为强硬的“必须解释字段清单”进行刚性约束，杜绝 AI 略过深层字段。
   - 设计了保留字段 Key 结构的 JSON 智能压缩与折叠算法，对长字符串和数组自动截断折叠以节省 Token。在保证报文结构合法无损的同时，将 Prompt 体积缩减 90% 以上，为 AI 生成详尽字段解释预留充足的 Output Token 空间。
   - 在 Prompt 中强化了对于具体枚举值（如订阅积分类型、会员等级、使用门槛等真实具体取值）的商业业务意义逆向剖析要求，杜绝字面翻译。
3. 请求与响应示例 JSON 强制保留：
   - 在 Prompt 硬性规则及文档结构要求中增加了严格限定，强制 AI 在 Markdown 输出的最末尾渲染完整高亮的请求/响应 JSON 压缩示例，避免其被遗漏。
