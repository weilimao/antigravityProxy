1. 修复长连接卡住 (Working)：将代理请求超时时间 (timeout) 从 15 秒调整为 5 分钟 (300,000ms)，避免因 LLMs 长时间停顿或大报告生成导致连接断开；
2. 修复 Brotli 报错解压失败：在 429 和 503 错误解析器中，添加对 br (Brotli) 压缩格式的支持，使其能正确解压并解析出 Claude 报错详情以触发配额刷新及重试逻辑；
3. 优化 Token 嗅探旁路流：将 Gemini 的 Token 嗅探逻辑彻底与客户端主数据流解耦，仅在 API 请求匹配 GenerateContent 且状态为 200 时通过事件旁路写入，且在旁路链路中加入 Brotli 支持，完全消除可能由于解压流卡顿导致的主流背压挂起隐患。
